---
title: "Install Beam"
---

<Tip>This should take less than 5 minutes!</Tip>

<AccordionGroup>
  <Accordion title="Create Your Account" icon={solid("circle-1")} defaultOpen="true">
	 * [Create a Slai account](https://www.slai.io/login) 
   * Grab your API keys from the [dashboard](https://www.slai.io/beam/apps/settings/api-keys).
  </Accordion>
  <Accordion
    title="Install Beam CLI"
    icon={solid("circle-2")}
  >
    In your terminal, run:
    ```bash
    curl https://raw.githubusercontent.com/slai-labs/get-beam/main/get-beam.sh -sSfL | sh
    ```
  </Accordion>
    <Accordion
    title="Add Your API Keys"
    icon={solid("circle-3")}
  >
    Run this in your terminal - you'll be prompted to paste in your API keys from Step #1
    ```bash
    beam configure
    ```
  </Accordion>
  <Accordion title="Setup Virtual Environment" icon={solid("circle-4")}>
		Set up a new folder wherever you want and create a virtual env:
		```curl
		python3 -m virtualenv .venv
		```
		Source it:
		```curl
		source .venv/bin/activate
		```
  </Accordion>
    <Accordion title="Install Beam SDK" icon={solid("circle-5")}>
		Last step! Install Beam SDK.
		```curl
		pip install beam-sdk
		```
    Now you're ready to start using Beam.
  </Accordion>
</AccordionGroup>
